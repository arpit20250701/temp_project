- Go to Secret Variables
- Go to Actions
- Add the following variable name and placeholder for secrets

        Secret: Access_Key_123

        Secret: Secret_Key_123
    Name: AWS_REGION
        Secret: AWS_Region_123
    Name: AWS_S3_BUCKET
        Secret: S3_123
